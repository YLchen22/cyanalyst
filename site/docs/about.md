# 关于 CyAnalyst

## 这是什么？

CyAnalyst 是一个专注于 **A 股市场** 的智能投研平台。通过自动化数据采集、量化分析和 AI 辅助，为投资者提供系统化的市场研究支持。

## 核心理念

> **用数据替代情绪，用系统替代直觉。**

我们不预测市场，而是提供多维度的数据视角和结构化的分析框架，帮助投资者做出更理性的决策。

## 数据来源

| 类型 | 来源 | 说明 |
|------|------|------|
| 行情数据 | 东方财富、腾讯财经 | 实时/历史行情、分时数据 |
| 财务数据 | 东方财富、同花顺 | 财报、业绩预告、分红 |
| 资金流向 | 东方财富 | 主力资金、北向资金、融资融券 |
| 新闻资讯 | 东方财富、腾讯财经 | 个股新闻、行业动态 |
| 全球市场 | 公开 API | 美股、港股、期货、外汇 |

## 免责声明

⚠️ **重要提示：**

本平台所有内容仅供研究参考，**不构成任何投资建议**。投资有风险，入市需谨慎。过去的表现不代表未来的结果。请根据自身风险承受能力做出独立判断。
